# Zerobase Mission1 과제

## 1. 개요
- Zerobase Mission1 과제
- Java와 JSP를 이용한 웹 페이지 개발
- 서울시 OpenAPI를 이용하여 주변의 WiFi 위치를 파악하는 웹페이지

## 2. 개발 환경
- Java 1.8 (openjdk1.8.0_361)
- Apache Tomcat 8.5.99
- MariaDB 11.3.1
